What is VICE?

VICE is a software tool constructed to organize, manage, and raise efficiency of robust firmware validation processes. This architecture will be implemented through a powerful and secure web application, utilizing cloud-based architectures.

Using VICE the user will be able to test existing and new hardware platforms while easily tracking the status of each platform. Register firmware model, brand, configuration, and debugging instrumentation is easily accessible through the VICE management tool.
